# Contexto de Proyecto: Pedro Carranza (Web Principal)

Este repositorio es la identidad digital de Pedro Carranza, Software Strategist e Ingeniero de Sistemas.

## Directrices Específicas
* **Perfil Profesional**: Destacar la trayectoria de más de 10 años en desarrollo y docencia.
* **Marca BitsaBots**: Se integra aquí como una redirección de blog centrada en modernización de sistemas legacy y APIs.
* **Stack**: Astro (SSG) y Tailwind CSS. El deploy se gestiona vía GitHub Actions.
* **Voz**: Analítica, ejecutiva y enfocada en soluciones de misión crítica (Pagos QR, ATC, Facturación).
